# QuoteVault

A beautiful inspirational quotes app with favorites management.

## Live Demo

**[quote-vault-phi.vercel.app](https://quote-vault-phi.vercel.app)**

## Features

- 60+ curated inspirational quotes
- Random quote generation
- Save favorites to localStorage
- View and manage saved quotes
- Copy quotes to clipboard
- Beautiful dark theme with gradient accents
- Smooth animations and transitions
- Mobile responsive design

## Tech Stack

- Next.js 16
- TypeScript
- Tailwind CSS v4
- Lucide React icons
- localStorage for persistence

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
